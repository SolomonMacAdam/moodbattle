# MoodBattle

MoodBattle is a Base mini app for onchain mood faction battles.

Users can choose a mood, release energy, boost any mood, reset their state, and watch the global power board update from the deployed contract.

## Overview

MoodBattle turns mood selection into a simple onchain competition.

Each mood acts like a faction.

Participants interact with the app through a connected wallet and submit actions to the Base contract.

The app reads the current state from the contract and displays the global mood power board.

## Features

- Choose a mood faction.
- Release energy for your selected mood.
- Boost any mood.
- Reset your current state.
